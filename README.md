# Rock-paper-scissors-game
a very simple and fun game 
This project is a Rock Paper Scissors game built using Python that runs in the command line interface (CLI).

In this game, the user plays against the computer. The user selects Rock, Paper, or Scissors, and the computer randomly selects one option. The program then compares both choices and determines the winner.

This project is useful for beginners to practice Python logic, random number generation, loops, and conditional statements.
